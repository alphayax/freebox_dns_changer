Freebox dns changer

Download as .zip Download as .tar.gz View on GitHub

Introduction

Ce projet permet de changer de maniere automatique les DNS de votre freebox. Les nouveaux serveurs DNS sont récupérés aléatoirement sur le site d'OpenNic. Le script peut etre croné pour mettre a jour regulierement les serveurs DNS

Installation

  1. Autorisez l'application sur votre freebox
  2. Accordez les droits à l'application dans l'interface web
